About

This is Noel Konagai's final project for the class called Mashups. The task was to create a web application that takes a user input, saves it to a server, and visualizes the collected data.

Where is home? was inspired by the diversity of New York University Abu Dhabi and Shanghai students' origin stories. Quite a few students have a list that extends beyond a single home. What's more interesting than the list is to see what each of these homes correspond to.

Is it the place they lived the longest? Is it the place where the person was born? What else? To capture each of these cases, the form includes a multiple choice selection. The instruction asks the client "Primarily, what do you consider this place to be?" When entering the responses, many ponder whether the location is home for them because they were born there? Or because their parents are from there?

In order to visualize the client input, I used WebGL Globe, which relies on three.js to create a 3D globe. In order to visualize the datapoints, the JSON data for WebGL Globe takes a longitude, a latitude, and a magnitude. Using Google Places API, the form geolocates and autocompletes the locations, when entered.

In the future, this web application hopes to collect data on how connected each individual feels to the location that they have entered. The client would enter a number from one to five, five meaning the highest level of connection. This web application also hopes to further visualize statistical insight into the locations that the clients have entered.
